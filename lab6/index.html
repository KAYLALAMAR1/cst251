Subject of This Lab

The subject of this lab is to create a new html file, add a form, and style it using a CSS file.

Challenges

I had trouble with adding my picture to the website. For some reason, the picture I downloaded from google drive and renamed to a jpg image wasn't working.

Results

After continuously checking that I entered the code correctly and the image still not working, I decided to try something else. I fixed this problem by taking a screenshot of the image and changing it from png to jpg. After I linked the new image file in my source code, it started working properly.
